# ReLeMesh 
Mesh Generator based on Deep Reinforcement Learning methods.


### Prerequisites

* Python 3.6
* TensorFlow 1..1
* Numpy 1.16.2
g++ 8.2.0

## Getting Started

To build the shared lib simply
```
mkdir lib && mkdir bin
make
```

Running the tests
```
make test
```

Running the python interface test:
```
make pyTest
```
